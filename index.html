    About Andowe.

    Andowe is a small, simple programming language I am experimenting with to learn more about how programming languages work. It is based originally on the Kaleidoscope language from the LLVM tutorials. The lexer, parser, and front end are all written in OCaml, while the back end uses LLVM for code generation.

    Building

    To build the latest version of Andowe, simply check out the head of this repository. Then, in your working copy, run

    $ make clean
$ make

    The resulting binary can be run with

    $ ./bin/andowe

    or

    $ ./bin/andowe path/to/code.and

    to run the code in a file on your machine.
